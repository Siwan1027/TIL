## 반복문

### if

- 조건이 참 True, 거짓 False 값에 따른 수행문을 설정함
    - 조건식의 boolean 값을 통해 실행됨

```bash
if [조건문] :
	[수행문]	
elif [조건문] :
	[수행문]
else :
 [수행문]
```

### while

- 조건이 참 일 경우 작업을 수행함
    - break: 구문을 통해 roop를 강제 종료 가능
    - 수행문 내에서 조건식에 영향을 끼치는 변수의 값을 변경시키는 경우가 많음

```python
while [조건식]:
	[수행문]
```

### for

- 조건문이 참인 범위 내에서 반복
    - 조건문 내에서 자동으로 변수의 값이 변경됨
    
    ```python
    for [조건식]:
    	[수행문]
    ```
